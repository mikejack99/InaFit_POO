# InaFit_POO
Projeto de conclusão POO

## Funcionalidades

- Calcula IMC
- Calcula idade da pessoa 
- Permite entrar com vários valores de dados como altura e peso
- Possui todos os conceitos ensinados em POO



## Como usar

Baixe o main.py e execute com uma IDE

linux:
```bash
  py main.py
```
